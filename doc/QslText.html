Qsl - Text

Anlegen einer Qsl-Textkonserve

Für jeden Landes Präfix kann ein QSL-Text in der Landessprache angelegt werden.
Je nach Rufzeichen-Präfix wird der zu druckende Text aus der Textdatenbank geholt und zum QSL-Druck herangezogen.
Vorausgesetzt es gibt eine Landestextkonserve. Ist das nicht der Fall nimmt das Druckmodul das default Land 'DEF' das ist englischer Text.

Die Textdatenbank besteht aus 8 Textspalten.
    1. Landes_Präfix
    2. Medium_Type ( L = Label, K = Karte )
    3 .OM_Anrede
    4. Begrüßung
    5. zum qso ( Kenner und andere Angaben )
    6. Text1
    7. Text2
    8 Text3 ( Büro:  ...... )

Diese Felder werden vom Druckmodul gelesen und je nach Mediumtype ( Label oder  Karte ) gedruckt.

Eine NEUE Landes-Textkonserve anlegen.
Im Druckmodule - Menü 'Configuriere Druckermedium' z.B. Label selektieren.

Es meldet sich der QSL Texteditor mit der Druckmaske für Label.
    Rechts im Feld `Landespräfix' den gewünschten Präfix eintragen.
    Es muss der Hauptlandespräfix genommen werden, z.B DL, G, CT usw.
    Dieser steht in der Dxcc-Länderliste. Sub-Präfixe dürfen nicht genommen werden.
    Jetzt den gewünschten Qsltext eintragen und den Button NEU drücken.

Eine weitere Möglichkeit.
    1. Einen bereits angelegten Mustertext laden. Im Fester 'Textkonserve für:' einen Präfix
        anklicken.
        Der Text für diesen Präfix wird geladen. Diesen Text jetzt als Muster verwenden und
        mit eigenen Wünschen überschreiben.
    2. Im Feld 'Landespräfix' den Präfix eintragen und NEU drücken.

Textkonserven können editiert und gespeichert werden.

Probedruck
Grundsätzlich sollte immer ein Probedruck gemacht werden. Papier muss hierbei nicht bedruckt werden. Drucke in eine Textdatei und schaue das Ergebnis mit einem PS-Dokumentenbetrachter an.
Alle Qsl Druckdateien stehen in (HOME)/tmp.

Hinweis: Zum Labeltext sollte natürlich auch eine passende Station-Beschreibung hinterlegt sein. Menü 'Betriebsarten' -> Rig.
