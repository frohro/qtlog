QRZ - Callbook

Ist eine QRZ-CD von www.qrz.com vorhanden, sollte diese installiert werden. Hiermit stehen mehr als 1 Mill. Rufzeichen im schnellen Zugriff zur Verfügung.

CD-Installation:

Unter dem Menü "Wähle" die Funktion "QRZ.db von der CD lesen" selektieren. Den Haken "Rufzeichen-Datenbank laden" setzen. Den Pfad zur CD setzen und "Daten Laden" drücken.
Es dauert etwas bis 1.1 Mill. Rufzeichen geladen sind.

Die Rufzeichen-Anzeige ist auf 500 Rufzeichen begrenzt. Man muss matchen. z.B. 'DL1' und 'cr'.
Wird in die Zeile geklickt werden alle Daten des OM's angezeigt.

Wer möchte kann auch gleich die 10.000 Bilder laden
Rufzeichen die ein Bild zur Verfügung stellen haben in der Spalte Image einen Stern, einfach rein klicken und das Bild ansehen,
