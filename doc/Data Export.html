Log-Daten Export im ADIF- oder CSV Format

Alle Daten des Logbooks  können exportiert werden. Die Zusammenstellung der Spaltenfelder ist frei wählbar.
Der Spaltennamen AWD ist ein Synonym für alle Award Typen.  DLD steht für DOK.
Der Awardtype wird gemäß der Einstellung im Reportdialog eingefügt.
Es werden zwei Query Beispiele mitgeliefert.

- Mit einem Klick auf einen Namen im Queryordner wird dieses selektiert.
  Es kann jetzt gemäß der Vorgabe im Query eine *.csv oder *.adi Datei erzeugt werden.
  Der Dateiname für die Outputdatei ist durch den Querynamen vorgegeben, kann aber überschrieben werden.

- Mit einem Doppelklick in die Query-Tabelle wird ein Query selektiert, geladen und
  zum Query-Editor umgeschaltet.

Query Editor
Links stehen die verfügbaren Feldnamen, rechts im Fester die ausgewählten Spalten Namen.
Mit einem Doppelklick werden die Namen aus- oder abgewählt,

Mit dem Button Übernehmen wird ein Query erzeugt und zurück geschaltet.
Das Query kann jetzt mit  Speichern gesichert oder unter einem neuen Namen gespeichert werden
und steht dann für späteren Bedarf zur Verfügung.

Mit den Button CSV oder ADIF wird die gewünschte Datendatei erzeugt und steht im HOME/xxx.csv oder *.adi zur Verfügung.

Wird der Output Dateiname gelöscht, erfolgt eine Rückfrage ob die Auswahl im Report angezeigt werden soll
