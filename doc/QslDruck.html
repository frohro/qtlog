QSL - Drucken

Beim Aufruf zeigt das Druckmodul alle QSOs mit gesetztem Printflag als Druckvorschlagliste.

Als erstes wird das Druckmedium festgelegt. Es gibt zwei vorkonfigurierte Medien.
Menü: Configuriere Druckermedium.
             Auswahl: Karte / Label
Hier werden die Medien : Label  oder die Standard Qslkarte angeboten.

- Unterstützt wird Label 4782 : 8 Label auf einem DIN A4 Blatt
- Standard Karte: 14 x 9 cm
Außerdem kann gewählt werden ob eine SWL Bestätigung gedruckt werden soll.

Wird in das Auswahlfenster geklickt, werden die Druck-Parameter selektiert und angezeigt.
Der Button  'Speichern' übernimmt die Parameter und schaltet den Druck-Dialog zurück.

Hat man sich für Label oder Karte entschieden wird als nächstes der Drucktext bearbeitet.

Menue: Configuriere Druckermedium
              Label
Es meldet sich die QSL-Textkonserve für Label. Der Anwender kann jetzt den Text für sein Label in einigen Bereichen selbst gestalten. 

Die Texte werden Landes-Präfix gesteuert gespeichert. Wird auf den Präfix DL geklickt erscheint als Vorschlag  ein Text in deutscher Sprache. Der Text kann individuell verändert und gespeichert werden.

Unter dem Präfix G kann englischer und unter F französischer Text hinterlegt werden, usw.
Soll ein Label für das Rufzeichen DL1HBD gedruckt werden wird die deutsche, für Rufzeichen mit G beginnend die englische Textkonserve herangezogen, usw.
Ist für ein Land kein Landestext hinterlegt wird die Konserve DEF genommen, dieser Text sollte englisch sein.

Es wurde versucht das Label- oder Karten-Layout nachzubilden.
Die editierbaren Felder sind beschreibbar, die farblich hinterlegten Druckbereiche werden mit Daten aus dem QSO-Datensatz gefüllt. Die Texte zwischen den < ... > Klammern sind Platzhalter.

Auf der rechten oberen Seite stehen die Dxcc-Prefixe und enthalten Textbeispiele.
Die Texte dürfen  nicht zu lang sein. Sie sollten mindestens auf das Papier passen.
Probedrucke sind dringend angesagt ! Um Papier zu sparen kann in eine xxx.ps Datei geduckt werden. Das Ergebnis kann mit z.B PS/PDF KGostView angesehen werden.


Drucken:
Ist die Druckvorschlagliste zu lang ( am Anfang ist sie immer zu lang ), werden QSOs durch einen klick in die Zeile aus der Liste entfernt. Beim nächsten Aufruf werden diese wieder zum Druck vorgeschlagen.
Die Selektion druckfähiger Qsos über das Rufzeichen ist ebenfalls möglich.

Beim Label kann  die Startposition auf dem Labelblatt zu positionieren werden.

Als letztes ist festzulegen ob Einzel- oder Sammelbestätigungen gedruckt werden sollen. Default ist die Einzelbestätigung.
Wird der Haken Sammelbestätigung gesetzt werden nur QSOs die mehrfach vorhanden sind angezeigt. Diese werden dann als sammelQSL gedruckt.

Nach Aufruf von 'Drucken' meldet sich der Printer. Hier ist nichts einzustellen mit Print geht es weiter.

Ist der Druck beendet erfolgt die Rückfrage ob der Druck ordnungsgemäß und ok ist. Falls ja: bekommen alle Qsos in der Druckliste den Vermerk 'Y'= Qsl bestätigt, das Printflag wird gelöscht.
Mit nein wurde lediglich Papier bedruckt - die QSOs bleiben als nicht geduckt in der Liste.

Mögliche Probleme:
Es kann vorkommen das in der Qso Tabelle der Eintrag Rig fehl. Der Druck wird abgelehnt.
